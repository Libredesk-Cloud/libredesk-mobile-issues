# Support

## Where to get help

| Channel | Use for |
|---|---|
| [GitHub Issues](https://github.com/Libredesk-Cloud/libredesk-mobile-issues/issues/new/choose) | Bug reports, feature requests, questions about the mobile app |
| [GitHub Discussions](https://github.com/Libredesk-Cloud/libredesk-mobile-issues/discussions) | General feedback, open-ended questions, sharing ideas |

---

## What to include when asking for help

When asking a question or reporting a problem, including the following information upfront will speed up the response significantly:

- **App version** — visible in the app settings or About screen
- **Platform** — Android or iOS
- **Device model and OS version** — e.g., iPhone 13, iOS 17.2
- **Libredesk server version** — if known, available in your admin panel
- **What you were trying to do**
- **What happened instead**
- **Any error messages or screenshots**

---

## What is not covered here

This repository covers the **Libredesk Mobile app** only. The following are outside its scope:

- Libredesk server installation, configuration, or upgrade issues
- Libredesk web interface bugs or feature requests
- General self-hosting infrastructure questions (reverse proxy, SSL, Docker, etc.)

For server-side or core platform issues, please refer to the upstream Libredesk project and its official support channels.

---

## Response time

This project is maintained by a small team. Response times may vary. Please be patient, and avoid opening duplicate issues or sending repeated messages.
