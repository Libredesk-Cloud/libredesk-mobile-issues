# Libredesk Mobile Issues

**Public issue tracker and release hub for Libredesk Mobile.**

---

## About Libredesk Mobile

Libredesk Mobile is the official mobile application for [Libredesk](https://libredesk.io) — an open-source, self-hosted customer support platform. The mobile app allows support agents to manage conversations, contacts, and notifications directly from their Android or iOS device.

---

## What this repository is for

This repository is the **public-facing hub** for Libredesk Mobile. It is the right place to:

- 🐛 **Report bugs** — unexpected behavior, crashes, UI issues
- 💡 **Request features** — suggest improvements or new capabilities
- 📋 **Track known issues** — see what is already identified and being worked on
- 📦 **Follow releases** — browse release notes, changelogs, and downloadable builds
- 💬 **Share feedback** — general thoughts on the mobile experience

Public releases may include:
- Android APK files for direct installation
- Detailed changelogs and release notes

> **Note:** The "Source code" archives automatically generated by GitHub on release pages contain only the content of this public repository — **not** the private mobile app source code.

---

## What this repository does not contain

- ❌ Flutter or Dart application source code
- ❌ iOS or Android native project files
- ❌ Backend, API, or server-side code
- ❌ Private build configurations or signing credentials

The mobile app source code is currently **private**. This repository exists solely as a public tracker and release hub.

---

## Scope disclaimer

This repository is focused on the **Libredesk Mobile app experience** only. Issues related to the Libredesk server, API, or web interface should be reported to the appropriate upstream project.

---

## Before opening an issue

Please do the following before submitting a new issue:

1. **Search existing issues** to avoid duplicates.
2. **Check the [CHANGELOG](./CHANGELOG.md)** to see if your issue was already addressed in a recent release.
3. **Update to the latest version** of the app if possible.
4. **Read [CONTRIBUTING.md](./CONTRIBUTING.md)** for guidance on writing a useful report.

---

## How to report a bug

Use the [Bug Report](./.github/ISSUE_TEMPLATE/bug_report.yml) issue template. Include as much detail as possible:

- App version
- Device model and OS version
- Steps to reproduce
- Expected vs. actual behavior
- Screenshots or screen recordings if relevant

---

## How to request a feature

Use the [Feature Request](./.github/ISSUE_TEMPLATE/feature_request.yml) issue template. Describe the problem you are trying to solve, not just the solution you have in mind.

---

## Releases

All public releases are published in the [Releases](https://github.com/Libredesk-Cloud/libredesk-mobile-issues/releases) section of this repository. Each release includes:

- A changelog summarizing what changed
- APK download for Android (where available)

Archived changelogs are also available in the [`changelogs/`](./changelogs/) directory.
